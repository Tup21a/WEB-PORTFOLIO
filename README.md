# WEB-PORTFOLIO

A web portfolio project featuring a TypeScript implementation of Tic-Tac-Toe (Tres en Raya).

## Getting Started

### Prerequisites
- Node.js
- TypeScript (`npm install -g typescript`)

### Running the Tic-Tac-Toe Game
1. Compile the TypeScript file to JavaScript:
```bash
npx tsc game.ts
```
2. Start the game:
```bash
node game.js
```
